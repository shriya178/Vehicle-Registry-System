🚗 Vehicle Registry System (C++)
📌 Project Overview

The Vehicle Registry System is a C++ project developed using Object-Oriented Programming (OOP) concepts. It allows users to register different types of vehicles, view all registered vehicles, and search for a vehicle using its Vehicle ID.

This project demonstrates the use of Encapsulation, Inheritance, Static Data Members, Constructors, Destructors, Getter & Setter Functions, and Arrays of Objects.

✨ Features
Add different types of vehicles
View all registered vehicles
Search vehicle by Vehicle ID
Count total registered vehicles
Menu-driven program
Uses multiple OOP concepts
